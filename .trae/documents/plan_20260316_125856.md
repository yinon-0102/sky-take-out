## 分页查询模块开发计划

### 需要修改的文件（共 4 个）：

#### 1. EmployeeMapper.java
**添加方法声明：**
```java
/**
 * 分页查询
 * @param employeePageQueryDTO
 * @return
 */
Page<Employee> pageQuery(EmployeePageQueryDTO employeePageQueryDTO);
```

#### 2. EmployeeMapper.xml
**添加 SQL 语句：**
```xml
<select id="pageQuery" resultType="com.sky.entity.Employee">
    <select>
        SELECT * FROM employee
        <where>
            <if test="name != null and name != ''">
                AND name LIKE concat('%', #{name}, '%')
            </if>
        </where>
        ORDER BY create_time DESC
    </select>
</select>
```

#### 3. EmployeeServiceImpl.java
**添加实现方法：**
```java
/**
 * 分页查询
 * @param employeePageQueryDTO
 * @return
 */
public PageResult pageQuery(EmployeePageQueryDTO employeePageQueryDTO) {
    // 1. 开启分页
    PageHelper.startPage(employeePageQueryDTO.getPage(), employeePageQueryDTO.getPageSize());
    
    // 2. 执行分页查询
    Page<Employee> page = employeeMapper.pageQuery(employeePageQueryDTO);
    
    // 3. 封装分页结果
    return new PageResult(page.getTotal(), page.getResult());
}
```

#### 4. EmployeeController.java
**修复 BUG（第 91 行）：**
```java
// 修改前：return Result.success();
// 修改后：return Result.success(pageResult);
```

### 技术说明：
- 使用 PageHelper 实现物理分页
- 支持按员工姓名模糊查询（可选条件）
- 按创建时间倒序排序
- 返回格式符合统一的 `Result<PageResult>` 规范
