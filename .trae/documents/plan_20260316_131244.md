## 代码回滚计划

### 回滚方案
使用 `git restore` 命令将所有修改的文件恢复到上一版本（HEAD）

### 需要回滚的文件（5 个）：
1. `sky-server/src/main/java/com/sky/controller/admin/EmployeeController.java`
2. `sky-server/src/main/java/com/sky/mapper/EmployeeMapper.java`
3. `sky-server/src/main/java/com/sky/service/EmployeeService.java`
4. `sky-server/src/main/java/com/sky/service/impl/EmployeeServiceImpl.java`
5. `sky-server/src/main/resources/mapper/EmployeeMapper.xml`

### 执行命令：
```bash
git restore sky-server/src/main/java/com/sky/controller/admin/EmployeeController.java \
            sky-server/src/main/java/com/sky/mapper/EmployeeMapper.java \
            sky-server/src/main/java/com/sky/service/EmployeeService.java \
            sky-server/src/main/java/com/sky/service/impl/EmployeeServiceImpl.java \
            sky-server/src/main/resources/mapper/EmployeeMapper.xml
```

### 说明：
- 这会丢弃所有未提交的修改，恢复到最近一次提交的状态
- `.trae/` 目录是未跟踪文件，不会受到影响
